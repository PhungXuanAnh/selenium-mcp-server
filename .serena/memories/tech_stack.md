# Tech Stack

## Core Technologies
- **Python**: >= 3.10
- **Selenium**: Web automation framework
- **FastMCP**: Model Context Protocol server framework
- **Click**: Command-line interface creation

## Key Dependencies
- `selenium`: WebDriver automation
- `click`: CLI argument parsing
- `mcp[cli]`: MCP server framework
- `undetected-chromedriver>=3.5.5`: Stealth Chrome driver
- `setuptools>=80.9.0`: Package building

## Development Tools
- **uv**: Package manager and virtual environment tool (recommended)
- **pip**: Alternative package manager
- **MCP Inspector**: Testing tool for MCP servers

## Browser Support
- Chrome browser (required)
- Supports both normal ChromeDriver and undetected-chromedriver

## Build System
- **hatchling**: Build backend
- **pyproject.toml**: Project configuration following PEP 518/621
